# SaturnOS

SaturnOS is a modern, high-performance operating system built from the ground up in C and ARM64 Assembly.

The project follows a modular kernel architecture and is designed for scalability, security, and maintainability. SaturnOS aims to evolve into a complete operating system ecosystem featuring its own kernel, device drivers, memory management, process scheduler, filesystem, networking stack, graphical desktop environment, and developer tools.

## Current Status

Version: 0.1.0

### Features
- ARM64 Kernel Boot
- Custom Bootloader
- PL011 UART Driver
- Kernel Console
- QEMU ARM64 Support

## Vision

To build a production-quality operating system that provides a reliable, secure, and extensible platform for future computing needs.
